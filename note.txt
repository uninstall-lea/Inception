127.0.1.1       lbisson
127.0.0.1       lbisson.42.fr www.lbisson.42.fr

Lancez les commandes ci dessous pour etre bien clean:
sudo docker stop $(sudo docker ps -q) (arrete tous les conteneurs en cours d execution)
sudo docker rm $(sudo docker ps -aq)  (supprime tous les conteneurs)
sudo docker rmi -f $(sudo docker images -q) (supprime toutes les images)
sudo docker system prune -af   (supprime tous les conteneurs, images, volumes... non reference ou arrete)
sudo docker volume rm $(sudo docker volume ls -q) (supprime tous les volumes)
